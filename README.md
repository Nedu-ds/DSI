# DSI
Update Systema

How to use it
```
git clone https://github.com/Nedu-ds/DSI.git
cd DSI
```


----VIRTUAL ENV-----

*execute as administrator*
```
$ apt install python3.8-venv
$ python3 -m venv env
$ source env/bin/active
```

--- Install --------

$ pip3 install -r requirements.txt

---- EXECUTE

$ python main.py
